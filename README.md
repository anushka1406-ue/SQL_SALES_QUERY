# 🗄️ SQL Employees Project

## 📌 About
This project contains PostgreSQL queries for an Employees database.

## 📂 Files
- `employees_project.sql` → All SQL queries

## 🛠️ Concepts Covered
- CREATE SCHEMA, TABLE, ALTER, DROP
- INSERT, UPDATE, SELECT
- WHERE, ORDER BY, GROUP BY, HAVING
- JOINS (LEFT, FULL, INNER)
- Subqueries, CTEs, Window Functions
- GRANT, REVOKE (User Permissions)

## 💻 How to Run
1. Open pgAdmin
2. Connect to PostgreSQL
3. Open `employees_project.sql`
4. Run the queries
